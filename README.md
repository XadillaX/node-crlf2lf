# CRLF TO LF

CLI to convert CRLF (M$ Win) to LF (Unix / Linux).

## INSTALLATION

```sh
$ [sudo] npm install -g crlf2lf
```

## USAGE

```bash
$ crlf2lf -h

Usage: crlf2lf <DIR> [options]

DIR     the directory to be CRLF2LFed.

Options:
   -r, --recur   whether the directory should be recurred.
```
